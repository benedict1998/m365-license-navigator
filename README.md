# M365 License Navigator

Button-driven Microsoft 365 licensing navigator for mapping licenses to features, prerequisites, upgrade paths, and source references.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Enable GitHub Pages using **Settings > Pages > Source: GitHub Actions**. The workflow is in `.github/workflows/deploy.yml`.

## Disclaimer

This tool is an internal licensing reference aid. Microsoft licensing changes frequently. Validate purchasing and contractual decisions against official Microsoft documentation or an authorized licensing provider.
